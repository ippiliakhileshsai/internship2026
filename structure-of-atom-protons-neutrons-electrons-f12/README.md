# Atom Explorer

An interactive educational web application that helps students understand the structure of atoms through simulations, visualizations, experiments, and quizzes.

## Project Name

**Atom Universe: Interactive Atomic Structure Learning Platform**

## Live Deployment

🚀 Live Project: https://dandurohitha.github.io/AtomExplorerF12/Atom-Explorer/


## Project Overview

Atom Explorer is designed to make learning atomic structure engaging and interactive. Users can explore protons, neutrons, and electrons, build atoms, perform virtual experiments, and test their understanding through quizzes.

## Modules

- Dashboard
- Atomic Lab
- Build an Atom
- Mass Lab
- Quiz

## Technologies Used

- HTML5
- CSS3
