<strong>Project Title</strong>

<strong>Diseases Causing Microorganisms</strong>

<strong>Description</strong>

Diseases Causing Microorganisms is an interactive educational web project that helps users learn about harmful microorganisms and the diseases they cause. The website presents health-related content in a simple and engaging way using multiple pages, interactive navigation, a quiz, and a simulation activity.

<strong>Problem Statement</strong>

Many students and general users find it difficult to understand the different types of disease-causing microorganisms, how they spread, the illnesses they cause, and the methods used to prevent them. Traditional learning methods may feel too theoretical or less engaging, which can reduce interest and understanding.

<strong>Solution</strong>

This project provides a user-friendly web-based learning platform that explains microorganisms such as bacteria, viruses, fungi, and protozoa. It organizes information into clear sections, allows users to explore disease details, includes prevention guidance, and reinforces learning through a quiz and an outbreak simulation.

<strong>Features</strong>

- Splash screen with automatic redirection to the home page
- Dedicated pages for microorganisms, diseases, prevention, quiz, simulation, about, and conclusion
- Educational content for bacteria, viruses, fungi, and protozoa
- Disease-specific information including symptoms, causes, transmission, and prevention
- Interactive microorganism selection flow
- Quiz section with multiple-choice questions
- Simulation activity for practicing prevention decisions
- Responsive design for easier use on different screen sizes

<strong>Tech Stack Used</strong>

- HTML5 for page structure
- CSS3 for styling, layout, and visual design
- JavaScript for interactivity, quiz logic, simulation flow, and dynamic content rendering

<strong>How to Run Project</strong>

1. Download or clone this project folder.
2. Open the project in any code editor such as VS Code.
3. Make sure all files remain in the same folder structure, including the `images` folder.
4. Run the project by opening `index.html` in a web browser.

For a smoother development experience, you can also use a local server such as the VS Code Live Server extension.

<strong>Conclusion</strong>

Diseases Causing Microorganisms is a simple and effective educational website designed to improve awareness about harmful microbes and public health safety. By combining informative content with interactive activities, the project makes learning more engaging, practical, and easy to understand.
