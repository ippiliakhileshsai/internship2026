# Spark Lab – Conductivity Tester

## Project Title

**Spark Lab – Conductivity Tester**

## Description of the Project

Spark Lab is an interactive virtual science experiment that helps students understand the concept of electrical conductivity in liquids. Users predict how many liquids will conduct electricity, perform a simulated experiment, and receive instant feedback based on their answers.

## Problem Statement

Learning electrical conductivity through traditional classroom methods can be abstract and less engaging. Students often lack opportunities to safely experiment with different liquids and observe their conductive properties in an interactive way.

## Solution (What the Project Does)

Spark Lab provides a gamified virtual laboratory where users can:

* Explore conductivity experiments in a safe digital environment.
* Predict the number of conductive liquids.
* View simulated experiment results.
* Receive scores and feedback based on their predictions.
* Learn scientific concepts through interactive participation.

## Features

* Interactive virtual conductivity experiment.
* Quiz-based prediction before the experiment.
* Animated and engaging user interface.
* Instant scoring and feedback.
* Multiple screens for navigation (Landing, Labs, Quiz, Experiment, and Results).
* Responsive design suitable for desktop and mobile devices.

## Tech Stack Used

* React
* TypeScript
* Vite
* Tailwind CSS
* HTML5
* CSS3
* JavaScript (ES6+)
* Lucide React (icons)

## How to Run the Project

1. Clone or download the repository.
2. Open the project folder in VS Code or any code editor.
3. Install dependencies:

   ```
   npm install
   ```
4. Start the development server:

   ```
   npm run dev
   ```
5. Open the local URL displayed in the terminal (typically `http://localhost:5173`) in your browser.
