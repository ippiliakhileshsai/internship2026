# Project Requirements

## Functional Requirements

- Users can enter energy consumption details.
- System calculates sustainability scores.
- Users can view recommendations.
- Dashboard displays results dynamically.
- System stores and retrieves user data.

---

## Non-Functional Requirements

- Responsive user interface
- Fast performance
- Easy usability
- Reliable data storage
- Scalable architecture

---

## Software Requirements

### Frontend
- React
- HTML5
- CSS3
- JavaScript
- Axios
- Chart.js

### Backend
- Python
- FastAPI
- Uvicorn
- Pydantic

### Database
- PostgreSQL

---

## Hardware Requirements

- Processor: Intel i3 or above
- RAM: 4 GB or above
- Storage: 10 GB free space
- Internet Connection

---

## Development Tools

- Git
- GitHub
- PostgreSQL

## Expected Outcome
The Smart Energy Advisor application helps users analyze energy consumption patterns, calculate sustainability scores, visualize results through dashboards, and recieve recommendations for environmentally responsible energy usage.
