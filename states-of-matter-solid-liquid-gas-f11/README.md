#  States of Matter — Interactive Particle Simulator

An interactive, browser-based particle simulation that lets you explore the four states of matter — Solid, Liquid, Gas, and Plasma — with real-time physics, substance lookup, multilingual support, and a built-in quiz.

---

##  Problem Statement

Physics concepts like states of matter and phase transitions are often taught through static diagrams and textbooks, making it difficult for students to intuitively grasp how particles behave under different conditions. There's a lack of free, accessible, interactive tools that visualize these concepts in real time.

---

##  Solution

 States of Matter — Interactive Particle Simulator provides a live particle simulation where users can manipulate temperature and pressure to observe phase transitions instantly. It supports substance-specific data (real melting/boiling points), multilingual labels, and a quiz mode — making it a practical learning tool for students at any level.

---

##  Features

- 🧪 **4 States of Matter** — Simulate Solid, Liquid, Gas, and Plasma with distinct particle behaviors
- 🌡️ **Temperature Control** — Drag a slider from -200°C to 6000°C and watch automatic phase changes
- 🌀 **Pressure Control** — Adjust from Vacuum to Extreme pressure and observe particle density changes
- 🔍 **Substance Explorer** — Search any substance (water, gold, chocolate…) to load its real phase transition data
- ⚡ **Auto Phase Detection** — Particles automatically shift state based on temperature thresholds
- 🎯 **Particle Count Control** — Adjust the number of simulated particles (10–120)
- 🧠 **Quiz Mode** — Test your knowledge in English, Hindi, or Telugu
- 🌐 **Multilingual UI** — Interface supports English, Hindi, and Telugu
- 📊 **Live Stats Bar** — Displays current state, temperature, pressure, and kinetic energy in real time
- 🎲 **Randomize & Reset** — Explore random configurations or reset to defaults

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (CSS Variables, Grid, Flexbox, Animations) |
| Logic | Vanilla JavaScript (Canvas API) |
| Fonts | Google Fonts — Space Grotesk, Orbitron, JetBrains Mono |
| Rendering | HTML5 `<canvas>` for particle simulation |

> No frameworks, no libraries, no build tools — runs entirely in the browser.

## 🚀 How to Run the Project

1. **Clone the repository**
```bash
   git clone https://github.com/tanishakolariya/states-of-matter.git
```

2. **Navigate to the project folder**
```bash
   cd states-of-matter
```

3. **Open the file in your browser**
```bash
   # Option 1: Just double-click index.html
   # Option 2: Use VS Code Live Server extension
   # Option 3: Run a quick local server
   npx serve .
```

4. **No installation or dependencies required** — it's pure HTML/CSS/JS.
