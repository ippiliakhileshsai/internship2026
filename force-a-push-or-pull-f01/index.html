🤔 What is a Force?

🤜

Push Force

A push is a force that moves things AWAY from you! Like kicking a soccer ball, pushing a swing, or closing a drawer.

🧲

Pull Force

A pull is a force that brings things CLOSER to you! Like opening a fridge door, pulling a wagon, or playing tug-of-war.

🧠 Key Facts & Secret Formulas

⚖️ Newton's 1st Law: The "Lazy" Rule!

Objects are inherently lazy! An object sitting still wants to stay still forever. An object moving fast wants to keep moving fast forever... unless a force pushes or pulls it! Both push and pull forces break this state of rest.

🚀 Newton's 2nd Law: The Power Formula!

The heavier an object is (its Mass), the harder you have to push it (Force) to make it speed up (Acceleration). This creates the most famous rule in physics:

Force = Mass × Acceleration
