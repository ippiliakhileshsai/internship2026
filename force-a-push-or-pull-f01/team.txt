PROJECT DESCRIPTION

This project demonstrates the concept of Force: A Push or Pull through an interactive simulation. Users can apply push and pull forces to objects and observe their movement. The simulation helps students understand how forces affect motion in everyday life.

MVP (Minimum Viable Simulation)

- Show one object (box/cart).
- Push button moves the object forward.
- Pull button moves the object backward.
- Display message:
  "Force is a Push or Pull."
- Simple animation showing object movement.

TEAM GOAL

To create an engaging educational simulation that helps learners understand the basic concept of force through interactive visual demonstrations.
