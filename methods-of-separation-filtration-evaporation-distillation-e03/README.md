# Methods of Separation Simulation

## Project Title

Methods of Separation: Interactive Science Simulation

## Project Description

Methods of Separation is an interactive educational web application designed to help students understand and visualize the different techniques used to separate mixtures. The project focuses on three important separation methods: Filtration, Evaporation, and Distillation.

Instead of learning only through textbooks, students can interact with animations, perform virtual experiments, listen to audio explanations, switch between languages, and test their understanding through quizzes and challenges. The project aims to make science learning more engaging, accessible, and enjoyable.

## Problem Statement

Many students find it difficult to understand separation techniques because these processes are often explained using static diagrams and theoretical descriptions. Traditional learning methods do not effectively demonstrate how these techniques work in real-life situations, making it challenging for students to visualize and remember the concepts.

## Solution

This project provides an interactive simulation platform that allows students to explore separation methods through animations and hands-on virtual experiments. Students can observe the process step-by-step, interact with laboratory equipment in a virtual lab environment, and reinforce their understanding through quizzes and progress tracking.

The simulation makes abstract scientific concepts easier to understand by transforming them into visual and interactive learning experiences.

## Features

* Interactive simulations for Filtration, Evaporation, and Distillation
* Virtual laboratory for performing science experiments
* Play, Pause, and Replay controls for animations
* Multilingual support (English and Telugu)
* Audio narration for improved accessibility
* Educational content with real-world examples
* Progress tracking system
* Achievement badges and rewards
* Interactive quiz section for self-assessment
* Responsive design for desktop and mobile devices

## REAL WORLD APPLICATIONS
Filtration is used in tea and coffee preparation. Evaporation is used in salt production. 
Distillation is applied in water purification, chemical industries, and alcohol production.

## How to Run the Project

### Method 1: Run Locally

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Open the project folder.

3. Open `index.html` in your browser.

### Method 2: Using VS Code Live Server

1. Open the project in Visual Studio Code.
2. Install the Live Server extension.
3. Right-click on `index.html`.
4. Select **Open with Live Server**.
