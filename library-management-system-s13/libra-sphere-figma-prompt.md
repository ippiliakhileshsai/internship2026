# Design a modern web application UI in Figma for a Smart Library Management System called:

# **"LibraSphere – Smart Library Management System"**

### Goal:

Create a modern, responsive web-based Library Management System that enables students, faculty, and librarians to efficiently search, issue, reserve, and manage books through a secure role-based platform with CAPTCHA authentication.

---

## Style

* Modern, clean, professional UI
* Academic and premium design
* Color palette:

  * Deep Navy Blue (#1E3A8A)
  * Sky Blue (#3B82F6)
  * White (#FFFFFF)
  * Light Gray (#F8FAFC)
  * Emerald Green (#10B981)
* Rounded cards
* Soft shadows
* Glassmorphism elements
* Modern typography (Poppins / Inter)
* Spacious layouts
* Professional illustrations
* Dashboard style similar to Notion, Google Workspace, and modern SaaS products
* Responsive desktop-first design
* Smooth animations and hover states
* Minimal but visually impressive
