HOW TO DEPLOY THIS HTML FILE (Explained Simply)
=================================================

We deployed the file "magnetic_simulation.html" (renamed to "index.html") 
as a live website using Vercel. Here's the step-by-step:

1. WHAT IS DEPLOYMENT?
   Deployment means taking a file from your computer and putting it 
   on the internet so anyone can access it via a URL.

2. WHAT WE USED — Vercel
   Vercel is a free platform that instantly turns static files (HTML, 
   CSS, JS) into a live website. No server setup needed.

3. STEPS TO DEPLOY:

   a) Install Node.js and npm (if not already installed)
      - Needed to run the Vercel CLI tool

   b) Install Vercel CLI
      $ npm install -g vercel
      Or simply use: npx vercel (runs without installing)

   c) Log in to Vercel
      $ npx vercel login
      This opens a browser link to authenticate your account.

   d) Navigate to the project folder
      $ cd /path/to/your/project

   e) Deploy
      $ npx vercel --prod --yes
      This uploads the file and gives you a live URL.

4. WHY WE RENAMED THE FILE
   Vercel (and most static hosts) look for "index.html" as the 
   default entry point. So we renamed:
     magnetic_simulation.html  →  index.html
   Now visiting the root URL (e.g., https://site.vercel.app) 
   serves the page directly.

5. THE LIVE SITE
   URL: https://swecha-ruby.vercel.app

6. ALTERNATIVE FREE HOSTING OPTIONS
   - GitHub Pages: Push to a GitHub repo, enable Pages in settings
   - Netlify Drop: Drag-and-drop the HTML file at netlify.com/drop
   - Python local server: python3 -m http.server (only works on 
     your own network)
