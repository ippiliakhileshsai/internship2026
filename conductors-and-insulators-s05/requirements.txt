CONDUCTOVerse - Conductors and Insulators Learning Website


PROJECT REQUIREMENTS


Project Name:
ConductoVerse - Interactive Learning Website for Conductors and Insulators

Project Objective:
To help school students learn about conductors and insulators through interactive content, experiments, quizzes, and virtual activities.

Target Users:

* School Students
* Teachers
* Science Learners

Technology Stack:

* HTML5
* CSS3
* JavaScript
* VS Code
* GitHub

FEATURES / MODULES

1. Home Page (index.html)

   * Introduction to ConductoVerse
   * Navigation cards
   * Start Learning button

2. Learning Module (learn.html)

   * Conductors explanation
   * Insulators explanation
   * Comparison table
   * Real-life examples
   * Fun facts section

3. Virtual Bulb Experiment (bulb.html)

   * Select material
   * Bulb glows for correct answer
   * Error message for incorrect answer

4. Quiz Module (quiz.html)

   * Multiple-choice questions
   * Score calculation
   * Performance badge

5. Real-Life Applications (uses.html)

   * House wiring
   * Power lines
   * Chargers
   * Safety gloves
   * Switch boards

6. Virtual Sorting Lab (lab.html)

   * Drag and drop activity
   * Conductors and Insulators classification
   * Score tracking

7. Results Module (results.html)

   * Student name input
   * Achievement certificate generation
