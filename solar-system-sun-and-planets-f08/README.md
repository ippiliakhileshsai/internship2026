# 🌌 CosmiCode

An interactive single-page web application that teaches children (8–14 years) about the Solar System through visuals, games, and quizzes.

---

## 🚀 Overview

CosmiCode presents Solar System concepts using:

* SVG illustrations
* Fun facts and badges
* Interactive navigation
* Drag-and-drop activities
* Quizzes with audio and visual feedback

---

## 📖 Story Architecture

**Intro → Overview → Sun → Inner Planets → Outer Planets → Planet Order Game → Quiz → Scoreboard**

---

## 🎯 Learning Objectives

* Identify the Sun and all eight planets
* Recall the order of planets from the Sun
* Differentiate between inner and outer planets
* Reinforce learning through games and quizzes
* Encourage curiosity about astronomy
